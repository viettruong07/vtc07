Nested controllers with model varibles defined directly on the scopes

(typing on an input filed, with a data binding to the model. overrides the same varible of a parent scope)

First controller

First name: {{firstName}}



Second controller (inside First)

First name (from First) {{firstName}}
Last name ( new varible): {{lastName}}
Full name: {{getFullName()}}




Third controller (inside Second and First)

First name (from First): {{firstname}}
Middle name (new varible): {{middleName}}
Last name (from Second): {{$parentlastName}}
Last name ( redefined in Third): {{lastName}}
Full name (redefined in Third): {{getFullName()}}
